# Implementation Notes

The active application is `apps/site/` — a Nuxt 4 + Vue 3 site that serves Pine Tar Sports Fund's marketing pages and investor deck system. Decks are YAML files edited no-code via Nuxt Studio.

`apps/dashboard/` (the original React + Vite deck builder) has been archived. See [`apps/dashboard/ARCHIVED.md`](apps/dashboard/ARCHIVED.md).
